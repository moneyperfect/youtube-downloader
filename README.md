# YouTube Video Downloader 油管视频下载器

[English](#english) | [中文](#中文)

---

## 🌐 English

### Features

- 🎥 **Video Download**: Download YouTube videos in multiple formats (MP4, WebM, MKV, etc.)
- 🎵 **Audio Extraction**: Extract audio only (MP3, AAC, FLAC, etc.)
- 🌐 **Proxy Support**: Built-in proxy detection and configuration
- 🎨 **Beautiful UI**: Elegant web interface with progress tracking
- 📊 **Format Selection**: Choose from available video/audio formats
- 🔄 **Batch Download**: Support for multiple video downloads

### Tech Stack

- **Backend**: Python (yt-dlp, Flask)
- **Frontend**: HTML, CSS, JavaScript
- **UI Design**: Custom ornate/elegant theme

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/moneyperfect/youtube-downloader.git
cd youtube-downloader

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the downloader
# Windows:
启动下载器.bat
# Or:
python backend/main.py
```

### Requirements

- Python 3.8+
- ffmpeg (for format conversion)
- yt-dlp

### Usage

1. Open your browser and visit `http://localhost:5000`
2. Paste a YouTube video URL
3. Select your preferred format
4. Click download and wait for completion

## 🇨🇳 中文

### 功能特点

- 🎥 **视频下载**: 支持多种格式下载（MP4、WebM、MKV 等）
- 🎵 **音频提取**: 支持仅提取音频（MP3、AAC、FLAC 等）
- 🌐 **代理支持**: 内置代理检测和配置功能
- 🎨 **精美界面**: 优雅的网页界面，实时显示下载进度
- 📊 **格式选择**: 多种视频/音频格式可选
- 🔄 **批量下载**: 支持多个视频批量下载

### 技术栈

- **后端**: Python (yt-dlp, Flask)
- **前端**: HTML, CSS, JavaScript
- **界面设计**: 自定义华丽风格主题

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/moneyperfect/youtube-downloader.git
cd youtube-downloader

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行下载器
# Windows 双击运行:
启动下载器.bat
# 或者在终端运行:
python backend/main.py
```

### 环境要求

- Python 3.8+
- ffmpeg（用于格式转换）
- yt-dlp

### 使用方法

1. 打开浏览器访问 `http://localhost:5000`
2. 粘贴 YouTube 视频链接
3. 选择你想要的格式
4. 点击下载并等待完成
